# Easy-Quant 量化策略库

Easy-Quant 是一个持续迭代的量化交易开源仓库。涵盖了从传统因子截面排序到机器学习非线性权重的探索过程。
目前代码适配 Supermind / MindGo API 规范。

## 版本演进记录

### V2: 十倍计划 (Project Decillion) - 2026.04
* **定位**: 主力纯多头轮动策略。
* **数据区间**: 2021.01 - 2026.04
* **回测表现**: 累积收益率 1111.99%，夏普比率 2.60，最大回撤 23.01%。
* **演进说明**: 
  * 早期尝试了叠加 HMM (隐马尔可夫模型) 进行宏观择时（见 `research_v2_hmm_explore.py`），回测发现大级别震荡市中存在严重状态漂移。
  * 最终版本弃用择时逻辑，改为扩充波动率因子、使用 LightGBM 进行非线性特征挖掘，并在头寸管理上采用基于预测得分的平方加权分配法。配合 12% 移动止损截断左侧风险（见 `research_v2_final.py` 与 `trade_v2_final.py`）。

### V1: 基础多因子框架
* **定位**: 量化管线基座。
* **演进说明**: 原有仓库基准代码，实现了基础因子计算、等权重调仓逻辑。归档于 `V1_Basic/`。
